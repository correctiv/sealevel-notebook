=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 27-Oct-2009
The Norwegian Mapping Authority have revised their datasets The benchmark reference for Hammerfest is now U3 N25 which is 8.655m above TGZ. As a result RLR was re-calculated using 2008 for the RLR year. The revised RLR factor is now 5.045m. RLR (2008) is 13.7m below TGBM U3 N25
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 29-Oct-2009
The PSMSL dataset begins in Jan 1957. Apparently there is earlier data on the Norwegian database but according to the site sheet the data before 1957 should not be used in any analysis.
=========================== FOLLOWING PSMSL DOCUMENTATION ADDED 04-Aug-2010
There is a datum problem with Hammerfest data for 1970. Because of this the data has been 'switched off' within the database. This period of data does not appear on the Norwegian website.
